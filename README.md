# 🚗 AutoGestão - Sistema de Cadastro de Veículos

O **AutoGestão** é uma aplicação web moderna desenvolvida para o gerenciamento de estoque de veículos. Este projeto foi criado como um desafio profissional acadêmico, integrando tecnologias de front-end e back-end para oferecer uma experiência premium e funcional.

![Preview do Projeto](https://raw.githubusercontent.com/brunokali/Car_veiculos/main/js/../car_veiculos_preview.png) *(Nota: Você pode adicionar sua própria screenshot aqui)*

## 🚀 Funcionalidades

- **Dashboard Inicial**: Visão geral e navegação intuitiva.
- **Cadastro de Veículos**: Formulário completo com validação de todos os campos obrigatórios.
- **Consulta em Tempo Real**: Listagem dinâmica de veículos consumindo dados via JSON do back-end.
- **Design Premium**: Interface baseada em Glassmorphism, Dark Mode e responsividade total.

## 🛠️ Tecnologias Utilizadas

- **Front-end**: HTML5, CSS3 (Variáveis, Flexbox, Grid), JavaScript (Vanilla/Fetch API).
- **Back-end**: PHP 8.x (Arquitetura orientada a APIs).
- **Banco de Dados**: MySQL / MariaDB (PDO para segurança contra SQL Injection).
- **Intercâmbio de Dados**: JSON.

## 📋 Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
- Um servidor local (XAMPP, WAMP, Laragon ou PHP/MySQL nativos).
- Git (opcional, para clonar o repositório).

## 🔧 Instalação e Uso

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/brunokali/Car_veiculos.git
   ```

2. **Configure o Banco de Dados:**
   - Acesse o seu gerenciador de banco de dados (ex: phpMyAdmin).
   - Crie um banco de dados chamado `car_veiculos`.
   - Importe o arquivo `database.sql` que está na raiz do projeto.

3. **Configure a Conexão:**
   - Se necessário, ajuste as credenciais de acesso ao banco no arquivo `php/config.php`.

4. **Inicie o Servidor:**
   - Mova a pasta do projeto para o diretório de arquivos do seu servidor (ex: `htdocs` ou `www`).
   - Acesse no navegador: `http://localhost/Car_veiculos`
